Longest Insequence Subsequence


Longest Increasing Subsequence is a way to find the longest increasing subsequence between all the number(or the select number) in the number list from beginning of the list to the end of the list, or any given start position in the list and end position in the list.

Is from left to right, find numbers in increasing order and increasing order only. And count the number as it goes, find the longest sequence it can have

Run time: O(n^2)

Data Structure :
array A[] for the list of numbers
array B[] for the longest increasing sequence up to that current position

Recursive Equation:
B[i] = { 1 + Max( B[j] : 0 <= j < i and A[j] < A[i]
{ 1

Code:
for ( int i = 0; i < A.length(); i++)
{
int maxN = 0;
for ( int j = 0; j < i; j++)
{
if( maxN < B[i] and A[j] < A[i])
{
maxN = B[i]
}
}
B[i] = maxN + 1
}

maxN is the maximum value of B[j]: 0 <= j < i and A[j] < A[i], if there is no j works, then maxN = 0
because of maxN is the biggest value of longest increasing sequence up until the current number, and A[j] < A[i], so B[i] is equals to 1 more than the longest increasing sequence before it.
